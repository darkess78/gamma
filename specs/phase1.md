# Phase 1 Spec

## Goal
Human speech -> STT -> LLM -> TTS -> spoken response

## Character target
Phase 1 should establish Shana's baseline conversational identity through Gamma's core loop.

## In scope
- one persona path
- one LLM provider
- one STT path
- one TTS path
- one local runner
- one API endpoint for text response
- one real-provider capability check for the chosen GPT path

## Out of scope
- deep memory
- tools
- avatar
- integrations

## Definition of done
A user can repeatedly talk to Shana and receive spoken replies with stable character behavior.
